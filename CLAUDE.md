# CLAUDE.md

이 파일은 Claude Code (claude.ai/code)가 이 저장소에서 작업할 때 참고할 수 있는 가이드를 제공합니다.

## 프로젝트 개요

상황별 성경구절을 보여주는 단일 페이지 웹 애플리케이션(SPA)입니다. 모든 애플리케이션 코드가 `index.html`에 포함되어 있으며, CSS와 JavaScript가 임베디드 형태로 작성되어 있습니다. 별도의 빌드 과정이나 외부 의존성이 필요하지 않습니다.

## 아키텍처

**단일 파일 구조**: 모든 HTML 마크업, CSS 스타일링, JavaScript 로직이 `index.html`에 포함되어 있습니다:
- **HTML (174-195줄)**: 카테고리 버튼과 구절 표시 컨테이너가 있는 메인 구조
- **CSS (7-172줄)**: 그라데이션 디자인, 애니메이션, 반응형 중단점을 포함한 스타일
- **JavaScript (197-359줄)**: 구절 데이터 객체와 이벤트 처리 로직

**데이터 구조**: `verses` 객체 (198-323줄)는 JavaScript 객체 리터럴로:
- 키는 한글 카테고리 이름 (예: "힘들때", "감사", "불안")
- 값은 구절 객체들의 배열이며, 각 객체는 `text`와 `reference` 속성을 포함
- 버튼의 `data-category` 속성은 이 키들과 정확히 일치해야 함

**UI 흐름**:
1. 사용자가 카테고리 버튼 클릭 → 카테고리 이름과 함께 `showVerse()` 함수 호출
2. 함수가 해당 카테고리 배열에서 무작위로 구절 선택
3. 선택된 구절 텍스트와 출처로 DOM 업데이트
4. "다른 구절 보기" 버튼으로 같은 카테고리 내 다른 구절 순환 가능

## 애플리케이션 실행

최신 웹 브라우저에서 `index.html`을 열기만 하면 됩니다. 서버, 빌드 단계, 의존성 설치가 필요하지 않습니다.

## 수정 방법

**새 카테고리 추가**:
1. `verses` 객체에 새 키를 추가하고 구절 객체 배열을 값으로 설정
2. HTML에 일치하는 `data-category` 속성을 가진 버튼 추가

**기존 카테고리에 구절 추가**:
1. `verses` 객체에서 해당 카테고리 찾기
2. `text`와 `reference` 속성을 가진 새 객체를 배열에 추가

**스타일 수정**:
- 메인 그라데이션 색상: `#667eea`와 `#764ba2` (16줄, 67줄)
- 모바일 중단점: 600px (149줄)
- 애니메이션: `fadeIn` (34줄), `slideIn` (97줄)
